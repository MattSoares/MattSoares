## Projects

### 🔥 FireWatch — IoT Wildfire Detection System

Early wildfire detection system built for **FIAP Global Solution 2026**. An Arduino-based device that monitors temperature, humidity and smoke in real time, classifies risk across 3 alert levels and outputs structured JSON ready for cloud/IoT integration.

> 220,000+ wildfire outbreaks recorded in Brazil in 2023. This was the starting point.

`Arduino Uno` `C++` `DHT22` `MQ-2` `LCD I2C` `ArduinoJson` `Wokwi` `HTML` `CSS` `JS`
